pnpm install -g @vue/cli 全局安装脚手架工具
pnpm update -g @vue/cli 更新

pnpm init @vitejs/app <自定义项目名> 用vite创建一个项目
cd 切换到项目中 然后pnpm i 安装依赖
再pnpm run dev 就可以了
